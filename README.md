# Dashboard Pró-Saúde 💵

## 📚 Informações sobre o projeto

* Desenvolvimento de um dashboard completo para gestão de medicina ocupacional, com controle de exames, empresas e funcionários. O sistema permite realizar vínculos entre colaboradores e empresas, gerar relatórios gerenciais e acompanhar o financeiro de cada exame com base em custo, valor e lucro, trazendo mais organização, automação e eficiência para a operação da clínica.

Site da empresa: [Pró-Saúde](https://www.prosaudema.com.br/).
&nbsp;

## 💻 Funcionalidades do projeto

* Cadastro de exames ocupacionais e complementares
* Gestão de empresas e funcionários
* Vinculação de colaboradores às empresas
* Controle e acompanhamento de exames
* Relatórios administrativos e financeiros
* Controle de custos, faturamento e lucro por exame
* Dashboard com visão geral operacional e financeira
